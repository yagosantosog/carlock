# **App Name**: CarLock Landing

## Core Features:

- Responsive Layout: Adapt the layout to different screen sizes and resolutions.
- Service Information: Provide clear and concise information about the vehicle tracking services offered by CarLock.
- Contact Form: Implement a contact form for users to request more information or to get in touch with the company.
- Customer Testimonials: Display customer testimonials to build trust and credibility.
- Pricing Plans: Present the pricing plans for the vehicle tracking services.

## Style Guidelines:

- Primary color: Vivid orange (#FF7C08) as the main brand color for highlights and calls to action.
- Background color: Very light gray (#F0F0F0) to ensure a clean and modern aesthetic.
- Accent color: Slightly desaturated, lighter orange (#FF9B42) for secondary accents and interactive elements, to provide visual interest while maintaining contrast.
- Font: 'Inter' sans-serif for both headlines and body text, ensuring readability and a modern feel.
- Use a clean, minimalist layout with plenty of white space to improve readability and focus on key information.
- Use simple, modern icons to represent key features and benefits of the vehicle tracking service.
- Incorporate subtle animations on scroll or when interacting with elements to enhance user experience without being distracting.
